# LethalLike

캐주얼한 **리썰라이크(Lethal Company-like)** 풍 게임 캡스톤 디자인을 위한
Unity 프로젝트 기본 폴더 구조를 구성한 저장소입니다.

## 프로젝트 구조

```text
LethalLike/
├── Assets/
│   ├── Animations/
│   ├── Art/
│   ├── Audio/
│   ├── Materials/
│   ├── Prefabs/
│   ├── Scenes/
│   ├── Scripts/
│   └── UI/
├── Packages/
│   └── manifest.json
└── ProjectSettings/
    └── ProjectVersion.txt
```

## 디자인 방향 (Capstone)

- 장르: VR 협동/탐사 기반의 캐주얼 호러-코미디 분위기
- 목표: 짧은 세션에서 리스크/리턴 판단과 팀 커뮤니케이션 경험 제공
- 핵심 루프: 탐사 → 수집 → 위험 회피/대응 → 귀환 및 점수 정산

## 시작 방법

1. Unity Hub에서 이 저장소 루트 폴더를 열기
2. 사용 버전 6000.3.10f1
3. `Assets/Scenes`에 메인 씬을 생성하고 기능 구현 시작
